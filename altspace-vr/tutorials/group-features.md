# <a name="using-the-groups-feature-to-create-private-events"></a>Utilisation de la fonctionnalité groupes pour créer des événements privés

Notre fonctionnalité de groupes offre aux hôtes la possibilité de créer un espace de VR ouvert à une liste approuvée d’invités. Les hôtes peuvent ajouter manuellement les noms d’utilisateur/courriers AltspaceVR auxquels ils souhaitent accorder l’accès à leur espace. Voici comment vous pouvez utiliser cette fonctionnalité pour héberger des événements, des réunions et des conférences :

> [!NOTE]
> Avant de pouvoir accéder aux « groupes », vous devez vous abonner aux fonctionnalités d’accès anticipé en vous connectant à altvr.com, en modifiant votre profil utilisateur sur le site Web et en sélectionnant accès en avant. ou à l’intérieur de l’application AltspaceVR, Paramètres, général, participez au programme d’accès anticipé.

1. Dans votre navigateur, accédez à  [https://altvr.com](https://altvr.com)   et connectez-vous à votre compte. 
2. Recherchez le menu déroulant **plus** .
3. Sélectionner des **groupes**

![Affichage Web AltspaceVR avec un menu déroulant plus grand en surbrillance](images/group-features-img-01.png)

4. Une fois dans la page groupes, vous pouvez voir une liste des groupes dont vous êtes membre ou un administrateur. 

![Page groupes d’affichages Web AltspaceVR](images/group-features-img-02.png)

5. Sélectionnez sur le bouton **créer** pour créer un nouveau groupe.
6. Renseignez les champs pour créer un groupe.

![Page de création de nouveau groupe avec champs d’entrée](images/group-features-img-03.png)

> [!NOTE]
> Vous pouvez pointer sur le «  ? » vert pour obtenir des informations supplémentaires sur les champs qui les contiennent.

7. Envoyez un e-mail à tous les participants et créez un compte AltspaceVR à l’aide de la même adresse e-mail. Ils peuvent créer un compte AltspaceVR dans le monde entier ou sur notre [site Web](https://account.altvr.com/users/sign_up).

## <a name="how-to-create-a-private-eventfrom-a-group"></a>Comment créer un événement privé à partir d’un groupe 

1. Une fois dans la page membre ou administrateur, sélectionnez le groupe auquel vous souhaitez accéder. 
2. **En tant qu’administrateur**, vous pouvez modifier toutes les options que vous avez configurées précédemment pour créer le groupe.Accédez au bouton « modifier » sur la page du groupe pour y accéder.  
3. **Ajout d’autres administrateurs pour un groupe :**   Lors de la modification, vous pouvez ajouter d’autres utilisateurs en tant qu’administrateurs pour faciliter la gestion du groupe.  
    * Il doit s’agir du nom d’utilisateur AltspaceVR pour l’utilisateur et non de son compte de messagerie.  
4. Sélectionnez **Créer un événement**.  

![Événement créé](images/group-features-img-04.png)

## <a name="how-to-attend-a-group-event-as-an-invitee"></a>Comment participer à un événement de groupe, en tant qu’invité 

1. Ouvrez le menu principal de l’application AltspaceVR : 

![Menu principal de l’application AltspaceVR](images/group-features-img-05.png)

2. Sélectionnez **événements** en bas du menu.
3. En haut, recherchez **mes événements**
4. Recherchez l’événement par son **nom** fourni par le créateur de l’événement dans l’affichage principal avec les images en mosaïque.Sélectionnez-la pour entrer l’événement s’il est en ligne.

![Application AltspaceVR avec le panneau mes événements mis en surbrillance](images/group-features-img-06.png)

## <a name="how-to-find-the-group-world-commons"></a>Comment trouver le monde du groupe ( ?) :  

1. Ouvrez le menu principal :  
2. Sélectionnez **mondes** en bas du menu.   
3. En haut, recherchez **mes mondes**.
4. Recherchez le nom de votre groupe et sélectionnez-le pour entrer dans le monde du groupe.

![Application AltspaceVR avec le panneau mes mondes mis en surbrillance](images/group-features-img-07.png)

## <a name="how-to-add-an-email-list-to-a-group"></a>Comment ajouter une liste de messages à un groupe 

Si vous avez un grand groupe et que vous devez les ajouter en bloc, vous devez d’abord créer une liste d’adresses de messagerie pour toutes celles associées à votre groupe. N’oubliez pas que s’ils s’inscrivent avec la liaison de compte Microsoft, vous aurez besoin de leur adresse de compte Microsoft pour que cette étape fonctionne.

### <a name="creating-a-list"></a>Création d’une liste

Utilisez un programme de feuille de calcul pour créer une colonne unique avec toutes les adresses de messagerie. Une fois cette opération effectuée, procédez comme suit :  

1. Créez ou modifiez votre groupe.  
2. Recherchez le champ **e-mails des membres** .  
3. Recherchez l’option **Choose file (choisir un fichier** ) pour importer vos courriers électroniques à partir d’un fichier * .csv.
4. Votre liste doit être importée avec un e-mail par ligne.

> [!NOTE]
> Si vous le souhaitez, vous pouvez également copier et coller des courriers électroniques dans le champ au lieu d’importer à partir d’un fichier * .csv.

### <a name="importing-a-list"></a>Importation d’une liste

Certaines plateformes peuvent vous permettre d’exporter un fichier * .csv (valeur séparée par des virgules) qui a une colonne « adresse de messagerie » dédiée.Vous pouvez sélectionner cette colonne pour l’utiliser.Une fois que cette opération est disponible, les étapes sont essentiellement les mêmes :  

1. Créez ou modifiez votre groupe.  
2. Recherchez le champ **e-mails des membres** .  
3. Recherchez l’option **Choose file (choisir un fichier** ) pour importer vos courriers électroniques à partir d’un fichier * .csv.
4. Votre liste doit être importée avec un e-mail par ligne.

> [!NOTE]
> Si vous le souhaitez, vous pouvez également copier et coller des courriers électroniques dans le champ au lieu d’importer à partir d’un fichier * .csv.

### <a name="im-having-trouble-managing-my-groups-what-do-i-do"></a>Je ne parviens pas à gérer mes groupes. Que faire ?

Si vous êtes membre ou administrateur d’un groupe, vous disposez d’outils supplémentaires pour gérer vos événements. Pour accéder à ces options, accédez à la page groupes du [site Web](https://account.altvr.com/groups).  

1. Sélectionnez membre ou administrateur dans la barre supérieure et sélectionnez le groupe que vous souhaitez gérer/mettre à jour. 
2. Une fois dans la page membre ou administrateur, sélectionnez le groupe auquel vous souhaitez accéder en cliquant sur son image de vignette.

![Volet membre ou administrateur ouvert](images/group-features-img-08.png)

Il s’agit de la page principale pour voir toutes vos appartenances à des groupes en tant que membre ou administrateur.Sélectionnez un groupe pour afficher ses options spécifiques.

![Écran appartenance au groupe](images/group-features-img-09.png)

La page de groupe s’affiche lorsque vous êtes membre d’un groupe.

**En tant que membre :**

* Vous pouvez accéder à la page d’accueil du groupe sur le site Web pour afficher les mises à jour du groupe. * vous avez un bouton qui permet de « conserver » le groupe.Une fois que vous avez obtenu, voici ce qui se produit :  
    * Vous n’aurez plus accès à la page d’accueil du groupe.  
    * Vous ne voyez plus ou n’entrez plus le groupe « Hangout ».  
    * Vous ne pouvez pas voir les événements de groupe.

![Page événements du groupe](images/group-features-img-10.png)

La page de groupe s’affiche lorsque vous êtes administrateur d’un groupe.Notez que les options d’administration s’affichent, quelle que soit la façon dont vous accédez à la page du groupe.

**En tant qu’administrateur :**   Vous avez les mêmes options qu’un membre, mais vous disposez de fonctionnalités supplémentaires.  

* Vous pouvez modifier toutes les options que vous avez configurées précédemment pour créer le groupe.Accédez au bouton « modifier » sur la page du groupe pour y accéder.  
    * **Ajout d’autres administrateurs pour un groupe :**   Lors de la modification, vous avez maintenant la possibilité d’ajouter d’autres utilisateurs en tant qu’administrateurs pour faciliter la gestion du groupe.  

> [!NOTE]
> Il doit s’agir du nom d’utilisateur AltspaceVR pour l’utilisateur et non de son compte de messagerie.  

* Vous avez également accès à un bouton **« créer un événement »** .  
    * **La création d’un événement à partir de la page du groupe définit automatiquement les autorisations de groupe pour l’événement.**  
    * Une documentation détaillée sur la création d’un événement via le site Web est disponible [ici](https://help.altvr.com/hc/en-us/articles/360033140854-Creating-an-Event).
* **Vous pouvez également faire disparaître le groupe en appuyant sur le bouton « supprimer ».**

> [!NOTE]
> Il s’agit d’une action destructrice, vous devez recréer le groupe si vous souhaitez le restaurer.

* Cette valeur est uniquement disponible pour le propriétaire du groupe, c’est-à-dire l’utilisateur qui a créé le groupe à l’origine.
* Elle n’est pas disponible pour les administrateurs du groupe.
